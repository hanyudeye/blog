* Key bindings

| Key binding | Description         |
|-------------+---------------------|
| ~<TAB>~     | Next link           |
| ~<BACKTAB>~ | Previous link       |
| ~H~ or ~[~  | Previous chapter    |
| ~L~ or ~]~  | Next chapter        |
| ~K/u~       | Scroll up           |
| ~J/d~       | Scroll down         |
| ~g m~       | Display metadata    |
| ~g r~       | Re-render document  |
| ~g t~       | Table of contents   |
| ~g v~       | View source         |
| ~g V~       | View content source |
